# School Logo & Multi-School Branding Setup Guide

## Quick Start: Adding Your School Logo

### Step 1: Access Settings
1. Log in as Principal/Admin
2. Go to **Dashboard â†’ School Settings & Branding**
3. Look for "School Logo" section

### Step 2: Upload Logo
1. Click **"Choose File"** button
2. Select your school logo (PNG or JPG)
3. Logo preview appears automatically
4. Add alt text for accessibility

### Step 3: Customize Colors
1. Click **Primary Color** picker
2. Select your school color
3. Repeat for Secondary Color
4. See live preview

### Step 4: Save
1. Click **"Save Settings"** button
2. Logo and colors now appear everywhere:
   - Dashboard header
   - Public survey pages
   - Student results pages
   - Reports

## Logo Requirements

| Requirement | Details |
|-------------|---------|
| **Format** | PNG or JPG |
| **Size** | Max 5MB |
| **Dimensions** | Any (auto-resizes) |
| **Background** | Transparent PNG recommended |
| **Logo type** | School seal, crest, or text logo |

**Scenario**: You manage 3 schools in your district

Option 1: One master admin with access to all
```

Sees: All school data
```

## Troubleshooting

### Logo Not Showing?
1. Clear browser cache (Ctrl+Shift+Del)
2. Check file size (<5MB)
3. Try PNG format (most reliable)
4. Refresh page after upload

### Colors Not Changing?
1. Use valid hex color (#RRGGBB)
2. Wait 10 seconds after saving
3. Clear browser cache
4. Check internet connection

### Can't Upload Logo?
1. Check file type (PNG/JPG only)
2. Reduce file size if >5MB
3. Check internet connection
4. Try different browser
